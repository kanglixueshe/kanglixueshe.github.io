Matrix 使用指南

Matrix 聊天室简介

Matrix 聊天室是一个与 XMPP 相似的聊天协议，同样是联邦式、开放协议的平台，有很多不同的服务器。除了主服务器 matrix.org 等知名服务器需要翻墙外，其他服务器很多。可以在此处选择服务器注册。尽量选择不需要填写邮箱的以确保隐私，如果是在要填写邮箱请填写 Proton 等匿名邮箱。我选择的是 gnulinux.club 不需要填写邮箱，直接在此注册。

Matrix 客户端

Matrix 客户端 有很多，标准实现是 Element，支持常见手机、电脑系统和浏览器。FluffyChat 是默菲推荐的手机端，不支持电脑（Linux 除外）。注册请从网页端直接注册。

注册流程及可能的问题解决

注册点击左下角 English 选项就可以切换成中文。

邮箱不填，直接点击注册，跳出来此对话框不填点继续。密码推荐使用 Firefox 的随即密码，Firefox 会为你随机生成并保存密码。当然如果没有 Firefox 不管。

如果点击继续后始终停留在下面的页面，请点击右键检查（Firefox 用户）或者使用 Ctrl+Shift+I（Chrome 用户），Firefox 用户会看到如下的错误（Content Security Policy），点击返回（即使知道一定会出错也不要省略这一步骤）

点击 Debugger 如图所示，找到 www.recaptcha.net，进入，如果排列在一行点击左下角的大括号，使其展开。找到类似的一行

po.src = 'https://www.gstatic.cn/recaptcha/releases/Ai7lOI0zKMDPHxlv62g7oMoJ/recaptcha__en.js';

在这一行后点击左侧行号会变蓝（设置了断点），再重复操作注册直到变灰。点击进入 console，输入刚刚 24 行的内容，注意把 .cn 改成 .com 其余不变，回车。然后点击右上的继续按钮即可完成。
